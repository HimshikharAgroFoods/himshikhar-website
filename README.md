# Himshikhar Agro Foods Website V2

## Open locally
Double-click `index.html`, or open it using the VS Code Live Server extension.

## Folder structure
- `index.html`
- `about.html`
- `products.html`
- `factory.html`
- `contact.html`
- `css/style.css`
- `css/responsive.css`
- `js/script.js`
- `images/`

## Publish free using GitHub Pages
1. Create a GitHub account.
2. Create a new public repository named `himshikhar-website`.
3. Upload every file and folder from this project.
4. Open repository **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch **main** and folder **/(root)**, then Save.
7. GitHub will provide the live website URL.

## Contact form
The contact form opens the visitor's email application with the enquiry details prepared. It does not require a paid server.
